# Pokédex Vue

Este projeto é uma Pokédex desenvolvida com [Vue.js 3](https://vuejs.org/), criada como parte de um desafio técnico para um processo seletivo de estágio na empresa **Maino**. A aplicação consome dados da [PokeAPI](https://pokeapi.co/) para exibir uma lista de Pokémon com scroll infinito, filtros e detalhes completos de cada criatura.

## 🔍 Funcionalidades

### Funcionalidades obrigatórias implementadas:

- ✅ **Listagem de Pokémon** com **scroll infinito**
- ✅ **Filtro por nome, ID, tipo e espécie**
- ✅ Página de **detalhes do Pokémon**, contendo:
  - Todos os **sprites** disponíveis
  - Lista de **movimentos de ataque**
  - **Cadeia evolutiva** completa
  - Todos os **jogos (game_indices)** em que o Pokémon aparece

### Funcionalidades diferenciais:

- ✅ **Responsividade**: layout adaptado para mobile e desktop
- 🚧 (em progresso) Internacionalização

## 🎨 Layout

A interface conta com:

- Barra de pesquisa com filtros dinâmicos
- Cards individuais para cada Pokémon com hover interativo
- Modal para exibição de informações detalhadas

## ⚙️ Tecnologias utilizadas

- [Vue.js 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [PokeAPI](https://pokeapi.co/)
- HTML5 + CSS3
- JavaScript (ES6+)
- Bootstrap 

## 💻 Como rodar o projeto localmente

1. Clone o repositório:

```bash
git clone https://github.com/matos-julio/pokedex-vue.git
cd pokedex-vue
```

2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

4. Acesse no navegador: `http://localhost:5173`

## ☁️ Deploy

A aplicação está publicada e disponível online através do Vercel:
